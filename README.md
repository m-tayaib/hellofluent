# 🌍 Language Learning Website – Design to Code

An interactive and modern language learning web platform built using React, TypeScript, and Tailwind CSS.

This project transforms a Figma UI design into a fully functional, responsive, and accessible web application. It focuses on clean architecture, scalability, performance, and user experience.

---

## 🚀 Tech Stack

- React
- TypeScript
- Tailwind CSS
- JavaScript (ES6+)
- HTML5
- Figma (Design Reference)

---

## ✨ Features

- Pixel-perfect Figma to Code implementation
- Fully responsive layout (Mobile, Tablet, Desktop)
- Interactive UI components
- Clean and scalable architecture
- Optimized performance
- Modern and accessible design

---
